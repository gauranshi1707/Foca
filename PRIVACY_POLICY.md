# Privacy Policy

Last updated: July 2026

Foca is a browser extension designed to help users stay focused while studying.

## Data Collection

Foca does not collect, transmit, sell, or share any personal information.

All user preferences, settings, study goals, and session statistics are stored locally on the user's device using the Chrome Storage API.

## Permissions

Foca requests only the permissions necessary to provide its functionality:

- Storage: Save user preferences and focus statistics.
- Alarms: Manage focus session timers.
- Notifications: Notify users when focus sessions end.
- Offscreen: Play session completion sounds.
- Host Permissions: Modify supported websites (such as YouTube and Instagram) to hide distracting content selected by the user.

## Third Parties

Foca does not send user data to any external servers.

# 🦭 Foca

### *Remove distractions. Keep what matters.*

A minimalist Chrome extension that helps students stay focused by removing distractions while preserving intentional learning.

## ✨ Why Foca?

Most website blockers force you to completely block YouTube or Instagram.

But what if you're actually trying to study?

Foca takes a different approach.

Instead of blocking learning resources, it removes the distractions around them.

Watch your lecture.

Read your notes.

Ignore everything else.

---

## 🚀 Features

### 🎥 YouTube

- Hide Homepage
- Hide Recommendations
- Hide Shorts
- Hide Comments
- Hide Sidebar
- Distraction-free learning experience

### 📷 Instagram

- Messages Only Mode
- Complete Block Mode
- Beautiful focus screen
- Quick Break system

### ⏱ Focus Sessions

- Custom focus duration
- Study goal tracking
- Session timer
- Session completion notifications

### 📊 Statistics

- Daily focus time
- Weekly focus
- Monthly focus
- Current streak
- Session history

### 🎨 Interface

- Dark Academia inspired design
- Minimalistic UI
- Smooth animations
- Clean typography

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/gauranshi1707/Foca.git
```

### Load into Chrome

1. Open `chrome://extensions`
2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the **Foca** project folder
5. Enjoy distraction-free studying.
